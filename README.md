### About

Nook is an isometric world building game I created cause I'm tired of all the games that focus on complex management systems. I built it to focus solely on the simple joy of creating and building

---

### Features

Explore it on your own by visiting https://nook-game.vercel.app :)

### Controls

| Action                | Control            |
| --------------------- | ------------------ |
| Move Camera           | Left Click + Drag  |
| Zoom                  | Mouse Wheel        |
| Place Object          | Left Click         |
| Select Multiple Tiles | Right Click + Drag |

### Getting Started

Clone the repository

```bash
git clone https://github.com/Khaiminh1902/Nook.git
```

Enter the project

```bash
cd nook
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
